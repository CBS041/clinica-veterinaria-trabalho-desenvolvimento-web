# 🐾 Sistema de Gestão para Clínica Veterinária

## 📋 Sobre o Projeto

O Sistema de Gestão para Clínica Veterinária foi desenvolvido como trabalho acadêmico da disciplina de Desenvolvimento Web, utilizando a arquitetura MVC (Model-View-Controller) para organizar e estruturar o código de forma eficiente.

O objetivo do sistema é auxiliar no gerenciamento das atividades de uma clínica veterinária, permitindo o cadastro e controle de informações relacionadas aos atendimentos, pacientes e demais processos administrativos.

---

## 🎯 Objetivo

Desenvolver uma aplicação que simule o funcionamento de uma clínica veterinária, aplicando conceitos de:

* Desenvolvimento Web;
* Arquitetura MVC;
* Programação Orientada a Objetos (POO);
* Organização em camadas;
* Boas práticas de desenvolvimento de software.

---

## 🏗️ Arquitetura Utilizada

O projeto foi desenvolvido seguindo o padrão MVC:

### Model

Responsável pela representação dos dados e regras de negócio do sistema.

Exemplos:

* Cliente
* Animal
* Consulta
* Veterinário

### View

Responsável pela interface com o usuário.

Funções:

* Exibir formulários;
* Mostrar informações cadastradas;
* Receber dados do usuário.

### Controller

Responsável por intermediar a comunicação entre Model e View.

Funções:

* Receber requisições;
* Validar informações;
* Executar operações do sistema;
* Atualizar a interface.

---

## ⚙️ Funcionalidades

### Cadastro de Clientes

Permite registrar os tutores dos animais.

### Cadastro de Animais

Permite armazenar informações dos pacientes da clínica.

### Gerenciamento de Consultas

Controle dos atendimentos realizados.

### Visualização de Dados

Exibição das informações cadastradas no sistema.

### Organização MVC

Separação das responsabilidades para facilitar manutenção e evolução do projeto.
